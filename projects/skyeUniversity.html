2019 design exercise

Option 2: Design an experience for students to discover orientation events and craft a visual system to accommodate different types of events: sports, music, visual arts, social groups, and volunteering events. Provide high-fidelity mocks for searching, browsing, and viewing the details for these different events.

For this design exercise, I chose to create an experience for students to discover orientation events. I have a strong interest in designing experiences, both virtual and physical, and I believed this was the best option to produce a project in both realms. Because of my background in architecture and the designing of physical spaces, I wanted this exercise to have a strong emphasis on the physical design of the orientation experience.

I structured the orientation experience around an exhibit-style format, where students can interact with a display in order to learn more about the various types of events during orientation week. In this display, I wanted the structure to encourage direct interaction with the exhibit, such as movement or a task-oriented display. Much of my inspiration stemmed from interactive museum exhibits, where users are encouraged to interact with a display to gain information.

Assumptions:

  • - University name: Skye University
  • - Location: N/A
  • - Length of Orientation: 1 week
  • - Each orientation event is open to all incoming students
  • - Each student has a Google educational email, and consequently has a Google Calendar
  • - Each student uses a Student Identification Card, and can communicate with a terminal via NFC

Display Station Sketches

After sketching and experimenting with a variety of ideas for the display form, I chose to represent the orientation schedule in a series of 5 panels, each one representing a different event type; sports, music, visual arts, social, and service. The screen would be housed in a rolling display, which can be moved by the user to each of the 5 panels, and would show the information for that specific type of event.

Color Study

After completing a rough black and white wireframe mockup, I investigated various color schemes for the color draft. I first considered using a bright warm color as the main accent shade, such as orange or red; however I ended up looking at more cool blue colors. Because blue is used to represent trust and peace, and also has a relationship to the name of the univerisity (Skye University), I looked at various ways to integrate the color into the design. I chose to move forward with a greyscale palette and a stone blue accent color.

Color Mockup V.2

After working with the Stone Blue color in the first color mockup, and after receiving user feedback, I decided to remove the dark grey header background. Though the header initially served as a visual anchor in the black and white mockup, the addition of the Stone Blue accent color made the heavy visual anchor unnecessary; instead, the dark header caused confusion and an unbalanced aesthetic.

Final Mockup

After inverting the header colors in the second color mockup, I added a visual distinction for events that were currently happening, and improved pages for the 'Print Reminder Note' and 'Send Event' pages.

In the event that a user is currently at the 'Sports' station and wants to see what other events are at that same time, they need to be able to see multiple event types on the orientation calendar. I decided to add an option on the Orientation Calendar page, which would allow a user to select which other event types they wish to see.

As a final addition to the mockup, I created an experiential prototype with InVision. The prototype includes animations of state changes, bringing proper attention to changes on the page, particularly within 'Browse Events' and 'Event Details'.

Sports Homepage

Sports Home Page

The home page is the front page of the orientation events programming. It provides an easy-to-read overview of the current day's events, with an option to browse all sports events or view the orientation calendar. It also draws attention to the special events that a student should keep in mind throughout the week. If the student already knows what they're looking for, a search option is also available.

Browse Events

Browse Events

Browse Events page provides an overview of the events during orientation week, along with the graphic assets or posters that may accompany such event. Because a poster or graphic can be a strong influence on a person's decision to attend an event, displaying each event's advertisement material was important. The page can also be sorted by type of event, or even by location.

Send event to my calendar

Event Details

With 'Send Event to my Calendar', the user can choose to send a specific event to their educational Google Calendar. This can be accomplished by tapping their student ID on the display's terminal, which recognizes their student ID number and email, allowing the system to send a Google invite for that event to the student's account.
