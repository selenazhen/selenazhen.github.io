Hedgy

HACKATHON ENTRY: PennApps 2017 [University of Pennsylvania]

Identity | Logo | Design | Frontend

Mobile Web App | 2017

Inspiration:

For this app, we decided to design a fun educational game for children. It caters to ages 4-8, where development and speaking skills are most important. In order to exercise these skills, HEDGY uses computer vision [Clarifai API] for identification of objects and combines it with linguistic accuracy [Google Cloud Speech API] to encourage healthy everyday education. The child identifies objects verbally, which is then cross-referenced using speech recognition and object recognition for accuracy in pronunciation and articulation.

In collaboration with Kristin Yin, Tiffany Zhu, and Christina Ou.
